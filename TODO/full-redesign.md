# Full Site Redesign

**Status:** PLANNED
**Priority:** MEDIUM

## Why

The current design was built incrementally. A ground-up redesign would:
- Establish a cohesive design language from scratch
- Implement modern editorial patterns (Bloomberg/Stratechery-level)
- Add interactive data visualization components
- Implement proper design tokens (not just CSS variables)

## Skills available for redesign

- `/frontend-design` — Anthropic's anti-generic-AI design skill
- `/taste` — Anti-slop, tunable density
- Playwright webapp-testing — Visual regression testing
- Frontend Design Toolkit — Figma-to-code workflow
- ui-skills (baseline-ui, accessibility, motion, metadata)
- design-auditor — 17-rule scoring

## Approach

1. Design audit with design-auditor (score current state)
2. Create design brief with `/frontend-design`
3. Build component library with taste-skill density controls
4. Implement with full Playwright visual testing
5. A/B test against current design
